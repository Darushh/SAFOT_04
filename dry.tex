\subsection*{Section A:}
\begin{enumerate}
    \item The symbol \('\) is necessary to prevent the interpreter from evaluating the arguments
    \(EXP\) and \(A\) before passing them to the \(EVAL\) function. \newline
    Without it, Lisp would evaluate \(EXP\) and \(A\) as variables and look for their values 
    but the \('\) symbol lets them pass as \(S-\)expressions.\
    \item The parameter \(A\) is the a list of key-value pairs so that \(EVAL\) knows to associate
    symbols to values in \(EXP\). \newline
    For example: \newline
    (eval 'x '((x . a) (y . b))) means Lisp will search \(A = \) ((x . a) (y . b)) for the value 
    of x and evaluate \(EXP = \) x accordingly which means \(eval\)'s return value will be a, 
    because that is the value for the key x in \(A\).
\end{enumerate}
